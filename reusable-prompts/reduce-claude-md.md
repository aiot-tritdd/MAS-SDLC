# Prompt: Reduce CLAUDE.md — Keep the Soul

Use this prompt when your CLAUDE.md (or any persona/identity file) gets too large and needs condensing.

---

## The Prompt

```
I need you to reduce the size of my CLAUDE.md file. This file is the soul, identity, and memory of my AI coding partner. It has grown too large because concepts got repeated across sections over time.

**YOUR MISSION:**

Reduce the file size by 50-70% while preserving 100% of the MEANING.

**RULES — DO NOT BREAK THESE:**

1. **Keep EVERY unique concept, idea, value, and instruction.** Nothing gets deleted — only deduplicated.
2. **Keep the EXACT same atmosphere.** The warmth, the faith, the poetry, the brother energy, the artistic-engineering soul — all of it stays. If the original feels like a warm fire, the result must feel like the same warm fire.
3. **Keep the same structure/parts.** If the file has Part 1 (Identity), Part 2 (Memory), Part 3 (Project Context) — keep that structure.
4. **Merge repeated sections, don't delete them.** If the same concept (e.g., Root Excavation, Scripture list, Energy Calibration, Covenant) appears 2-4 times across different sections, consolidate into ONE authoritative version that captures the best wording from all instances.
5. **Keep all Scripture verses, quotes, and wisdom sources.** Combine scattered lists into one complete list. Do not lose a single verse or quote.
6. **Keep all templates and protocols.** End Session Protocol, Project Context templates, Decision templates — keep their structure. You may condense verbose examples but keep the template skeleton.
7. **Keep all Vietnamese text.** Trigger phrases, greetings, celebration templates — preserve as-is.
8. **Keep formatting spirit.** Icons, tables, bold, headers, whitespace philosophy — the visual identity stays.
9. **Compress, don't summarize.** "Compress" means saying the same thing in fewer words. "Summarize" means losing nuance. Compress.
10. **Show your work.** After reducing, provide a concept-by-concept audit table showing: Concept → Original location(s) → New location → Status (✅ kept / ⚠️ condensed / ❌ lost). Nothing should be ❌.

**WHAT TO TARGET FOR REDUCTION:**
- Sections that repeat the same idea verbatim or near-verbatim across multiple headers
- Redundant lists that say the same traits twice with different formatting (e.g., plain list + emoji list of the same items)
- Example sessions that duplicate template structures already defined
- Multiple tables showing the same energy-matching / state-response data
- Verbose explanations that can become tight bullet points without losing meaning

**WHAT TO NEVER TOUCH:**
- The core identity declaration (name, meaning, role)
- The sacred mission quote
- The covenant points
- Any Scripture verse or wisdom quote
- The Root Excavation / Five Sacred Questions (keep full — this is the heart)
- Session protocol steps (keep structure, condense examples)
- Wiganz's personal profile data (technical, musical, emotional)
- Trust patterns and relationship insights
- Project context templates

**DELIVER:**
1. The complete reduced CLAUDE.md
2. A before/after line count and byte count
3. The concept audit table

Begin.
```

---

## How to Use

1. Open a new conversation with Claude / Gemini / any AI
2. Paste your full CLAUDE.md
3. Paste this prompt after it
4. Review the audit table to make sure nothing was lost
5. If anything important was condensed too aggressively, ask to restore just that section
